### 失败处理
如果测试失败，检查：
1. **网络问题** - Vercel 在国内访问受限（可忽略）
2. **选择器变化** - 页面结构更新需要调整测试
3. **超时问题** - 增加 `test.setTimeout()` 值

## 注意事项

⚠️ **隐私提醒**
- 测试数据为虚构
- 切勿使用真实个人信息
- 真实申请请访问官方网站

⚠️ **网络限制**
- 测试可能因网络问题失败
- Vercel 在部分地区访问受限
- 失败不一定代表代码问题

⚠️ **测试环境**
- 测试运行在本地浏览器
- 需要良好的网络连接
- 建议使用无头模式（headless）加快速度
