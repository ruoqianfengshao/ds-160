## 🔍 故障排查

### 常见问题

1. **注册失败 "Email already registered"**
   - 解决：使用新的邮箱或清理数据库测试数据

2. **页面加载超时**
   - 解决：检查 Vercel 部署状态，确认应用正常运行

3. **数据不持久化**
   - 解决：检查数据库连接，查看 API 请求是否成功

4. **登录后立即退出**
   - 解决：检查 JWT token 是否正确设置
