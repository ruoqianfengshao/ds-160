# Icon Placeholders

由于这是文本环境，图标需要手动创建。以下是图标规格：

## 需要的图标尺寸

1. **icon16.png** - 16x16 pixels
2. **icon48.png** - 48x48 pixels  
3. **icon128.png** - 128x128 pixels

## 设计建议

### 主题
- 美国国旗元素（红白蓝配色）
- 护照/文档图标
- 简洁现代风格

### 配色方案
- 主色：#667eea（紫色）
- 辅色：#764ba2（深紫）
- 强调色：#10b981（绿色）

### 图标内容建议
- DS-160 文字
- 或护照图标
- 或表单图标

## 快速生成图标

可以使用以下在线工具生成：
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- Figma / Sketch / Illustrator

## 临时替代方案

在图标文件创建之前，扩展仍可正常工作，只是工具栏图标会显示为默认样式。
