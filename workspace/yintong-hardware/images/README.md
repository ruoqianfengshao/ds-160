## 图片制作建议

1. **轮播图**：可以用公司厂房、设备、团队照片
2. **产品图**：白色背景，清晰展示产品细节
3. **证书**：营业执照、质量体系认证等扫描件
4. **导航图标**：可以用 iconfont.cn 下载或使用设计工具制作

如需要帮助制作图片，可以联系设计师或使用在线设计工具（如 Canva、稿定设计等）。
