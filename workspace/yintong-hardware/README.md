# 慈溪银通五金有限公司 - 微信小程序

## 📱 项目简介

这是一个纯展示型企业官网微信小程序，适用于五金零配件制造行业。

**功能模块：**
- ✅ 首页 - 公司形象展示、核心优势、快速导航
- ✅ 产品中心 - 产品分类展示、一键致电询价
- ✅ 关于我们 - 公司简介、厂房设备、资质证书
- ✅ 案例展示 - 合作客户、产品应用案例
- ✅ 联系我们 - 地址电话、地图导航、一键拨号

**技术特点：**
- 原生微信小程序开发
- 无需后端，数据写死在代码中
- 支持一键拨号、地图导航、分享功能
- 响应式设计，适配各种屏幕

## 🚀 快速开始

### 1. 准备工作

1. 注册微信小程序账号
   - 访问 https://mp.weixin.qq.com
   - 选择"小程序"注册
   - 准备材料：营业执照、法人身份证、对公账户

2. 下载微信开发者工具
   - 访问 https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
   - 根据操作系统下载安装

3. 获取 AppID
   - 登录小程序后台
   - 进入"开发" > "开发管理" > "开发设置"
   - 复制小程序 AppID

### 2. 导入项目

1. 打开微信开发者工具
2. 点击"+"添加项目
3. 选择项目目录：`yintong-hardware`
4. 填入你的 AppID
5. 点击"导入"

### 3. 修改配置

编辑 `app.js`，修改以下信息：

```javascript
globalData: {
  companyName: '慈溪银通五金有限公司',
  phone: '0574-XXXXXXXX', // 改为实际电话
  address: '浙江省宁波市慈溪市槐树路 282 号',
  latitude: 30.1667, // 可精确到具体纬度
  longitude: 121.2444 // 可精确到具体经度
}
```

**如何获取精确经纬度？**
- 方法 1：用高德地图/百度地图搜索公司地址，右键点击位置查看坐标
- 方法 2：用微信开发者工具的地图组件预览功能获取

### 4. 准备图片资源

参考 `images/README.md` 准备图片文件。

**临时测试可以不放图片**，页面会显示占位提示。

### 5. 编译预览

1. 在微信开发者工具中点击"编译"
2. 在模拟器中预览效果
3. 用手机微信扫码测试（需绑定为体验者）

### 6. 上传发布

1. 点击右上角"上传"按钮
2. 填写版本号和备注
3. 登录小程序后台
4. 进入"版本管理"
5. 提交审核
6. 审核通过后发布上线

## 📝 内容修改指南

### 修改公司信息

所有公司信息集中在 `app.js` 的 `globalData` 中统一管理。

### 修改产品列表

编辑 `pages/products/products.js`，修改 `products` 数组：

```javascript
products: [
  {
    id: 1,
    name: '产品名称',
    description: '产品描述',
    category: '产品分类',
    image: '/images/product1.jpg'
  },
  // 添加更多产品...
]
```

### 修改产品分类

编辑 `pages/products/products.js`，修改 `categories` 数组：

```javascript
categories: ['螺丝', '螺母', '冲压件', '车削件', '其他']
```

### 修改公司简介

编辑 `pages/about/about.js`，修改 `companyIntro`。

### 修改厂房设备列表

编辑 `pages/about/about.js`，修改 `equipment` 数组。

### 添加案例

编辑 `pages/cases/cases.js`，修改 `cases` 数组。

## 🎨 样式定制

### 修改主题色

全局主题色为蓝色 `#1E88E5`，如需修改：

1. 编辑 `app.wxss` 中的 `.btn-primary` 背景色
2. 编辑 `app.json` 中的 `navigationBarBackgroundColor`
3. 编辑各页面的 `.wxss` 文件中对应的颜色

### 修改字体大小

在 `app.wxss` 中调整全局 `font-size`。

## ⚠️ 注意事项

1. **电话号码格式**：不要包含空格或特殊字符，如 `0574-12345678` 需改为 `057412345678`

2. **图片尺寸**：建议按照 `images/README.md` 中的尺寸准备，过大会影响加载速度

3. **地图精度**：建议使用精确的经纬度坐标，否则导航可能不准确

4. **小程序审核**：
   - 确保营业执照经营范围包含相关内容
   - 联系方式必须真实有效
   - 图片不能有侵权问题

5. **版本更新**：修改内容后需要重新上传、提交审核、发布
