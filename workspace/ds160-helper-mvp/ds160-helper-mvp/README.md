# DS-160 Helper MVP

美国签证 DS-160 申请表辅助填写工具

## 技术栈

- **框架**: Nuxt 3
- **UI**: Vue 3 + TypeScript
- **样式**: Tailwind CSS v4
- **状态管理**: Pinia

## 功能特性

### Phase 0.5 MVP 功能

- [x] 12步引导式表单填写
- [x] 每个字段的中文说明和填写示例
- [x] 高风险字段橙色警示
- [x] 实时数据校验
- [x] 草稿自动保存（localStorage）
- [x] 中断恢复功能
- [x] 模拟同步功能（3次免费体验）
- [x] 响应式设计，移动端适配

### 页面结构

**SSR 页面（营销）**
- `/` - 首页
- `/features` - 功能介绍
- `/pricing` - 定价页

**CSR 页面（应用）**
- `/app/dashboard` - 用户面板
- `/app/drafts` - 草稿箱
- `/app/form/step-[1-12]` - 表单填写页
- `/app/form/step-12-confirmation` - 完成确认页

## 表单步骤

1. **个人信息** - 姓名、性别、出生日期等
2. **护照信息** - 护照号、有效期等
3. **联系方式** - 地址、电话、邮箱
4. **旅行信息** - 目的、行程安排
5. **旅行伙伴** - 同行人信息
6. **美国联系人** - 在美联系人或酒店
7. **家庭信息** - 父母、配偶信息
8. **工作/教育** - 职业、教育背景
9. **安全问题（1）** - 犯罪记录相关
10. **安全问题（2）** - 恐怖主义相关
11. **以往旅行** - 美国入境历史
12. **照片确认** - 上传签证照片

## 数据存储

- MVP 阶段使用 localStorage 存储草稿数据
- 不上传任何个人信息到服务器
- 数据仅保存在用户浏览器本地

## 高风险字段

以下字段会用橙色警告框标注：
- 政府职务相关
- 犯罪记录
- 违法行为
- 恐怖主义关联
- 毒品相关
- 以往拒签历史
- 移民申请历史
